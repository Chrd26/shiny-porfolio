Introduction

C++ is a high-level general purpose programming language created by Danish computer scientist Bjarne Stroustrup

as an extension to the C programming language or "C with classes".

C++ was designed with systems programming and embedded, resource-constrained software and large systems in mind with

performance, efficiency, and flexibility of use as its design highlights.

Hello World using C++

How to write a "Hello, world!" program using C++
        
#include <iostream>
int main()
{
std::cout << "Hello, world!\n"; 
}

C, C++, and C#

What is the difference?

C and C++ are pretty similar language since C++ is an extension for the C language. This is not true the case of C# which has similarities to the java programming language.

Here are some differences between these 3 languages:

  1. C++ can run C code but the reverse is impossible.
  2. C++ is an object oriented programming language, C# is a component-oriented language, and C is a paradigm-free language.
  3. C uses functions for processing but C++ uses objects.
  4. C# cannot use pointers unlike C++ and C. C# doesn't need to manage memory and it can manage it automatically.
  5. C#'s syntax is more forgiving than C and C++ which makes it easier to learn than the other 2.

Variables

Variables are objects that are used to use certain type of data.

To declare a variable, you must always define its type name and value. It is not needed to assign a value at definition. It is considered a good practice to name a variable using camelCase which helps to identify a variable while reading the code. There are multiple types of data that can be stored in a variable.

Types of Data
  • Integer data that are defined using int
  • Character data that is defined by using char
  • Floating data that is defined using float
  • Double floating data types that are defined using double

How to declare a variable

        
data-type name = value

Example:
int number = 5;

Variables can be used to input data for processing and store it. A variable with no value is a null variable. It is considered a good practice to initialize the variable which means to add value to it to avoid problems that may caused by an empty variable. For example, a null variable cannot be incremented by 1 since there is no value in it. It is possible to create a variable by using the auto data type. This automates the data definition process and assigns the right data type to the variable. This is only possible if the variable is initialized in during declaration and not later.

Example

        
#include <iostream>
int main()
{
auto number = 5;

//This declaration and initialization
//makes use of the auto type.
//The compiler will assign automatically 
//the right data type to the variables
//which in this case is int.
}

Conditionals

Conditional statements are programming language commands used to handle decisions and the flow of the code. They are used to execute different blocks of code whether the defined Boolean is true or false. There are 4 types of statements


  1. if statement

    2. These are the simplest type of conditional statement. If the condition is true, then the code block in the statement will run.

              
#include <iostream>

int main()
{
if (condition)
{
std::cout << "Hello!\n";
}
}
  2. if else statement

    3. This type of statement that lets the program run different blocks of code based on a certain condition. If the Boolean condition is true, then the code in the if's brackets is used. Otherwise, the program uses the code in the else's curly brackets.

              
#include <iostream>

int main()
{
if (condition)
{
std::cout << "Hello\m"; 
} 
else 
{ 
std::cout <<"Hi!\n"; 
}
}
  3. if - else if - else statement

    4. This is the third type of statement that uses the if keyword. This is a type of statement that uses multiple conditions. The else if statement can be used multiple times for a variety of conditions.

            
#include <iostream>

int main()
{
if (condition)
{
std::cout << "Condition 1 is true!\n";
}
else if (condition)
{
std::cout << "Condition 2 is true!\n";
}
else if (condition)
{
std::cout << "Condition 3 is true!\n";
}
else
{
std::cout << "All the conditions are false!\n";
}
}
  4. switch statement

    5. A switch statement is quite different than the if one and its variants. This is a great way if the condition has multiple values and the programmer wants to implement different logic for those certain values. The syntax of the switch statement is quite different too! A switch contains multiple cases and a case has its very own block of code that run if the condition is met. It might also contain a break; keyword. The break keyword tells the switch to stop checking the rest of the cases if the condition is true. It should also have a default case which will be used if no conditions are met similarly to the else statement. 


            
#include <iostream>

int main()
{
switch condition
{
case value1:
std::cout << "Condition 1 is true!\n";
break;

case value2:
std::cout << "Condition 2 is true!\n";
break;

case value3:
std::cout << "Condition 3 is true!\n";
break;
         
case value4:
std::cout << "Condition 4 is tru!\n";
break;

default:
std::cout << "The are not any conditions that are true!\n";
break;
}
}

Loops

Loops are used to run certain blocks of code multiple times. They can be used to iterate through arrays, to wait for certain inputs and apply the same code multiple times! There are 3 types of loops, while, do/while and for loop.


  1. While Loop

    2. A while loop repeats until its condition is met or can repeat forever. It can be used to make the program wait for certain inputs, repeat processes without stop or repeat parts of code until the right conditions are met.

    
#include <iostream>

int main()
{
while(true)
{
std::cout << "This code loops forever!\n";
}
}

#include <iostream>

int main()
{
while(x < 10)
{
std::cout << "This code loops until the x is bigger 10!\n";
x++
}
}

  2. Do/while loop

    3. This is quite similar to a while loop. The only difference here is that it lets the code to be used by program at least once.

    
      #include <iostream>

int main()
{
do
{
std::cout << "This code will run at least once!\n";
}
while(condition);
}

  3. For Loop

    4. This type of loop is used if we want our code to repeat certain amount of times! A great use of it is to iterate through an array. It uses the keyword for and typically is followed by a parenthesis that contains a declaration with an initialization, a conditional statement and an operation.

    
#include <iostream>

int main()
{

for (int i = 0; i < 10; i++)
{
std::cout << "This for loop has repeated << i << " times so far!\n""; 
}
}
